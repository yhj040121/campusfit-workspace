# 青搭

校园穿搭社交电商导购 APP 高保真原型，面向大学生垂直场景，覆盖内容信息流、发布编辑、社交互动、电商导购、校园标签体系、个人中心、搜索筛选等核心模块。

## 打开方式

直接在浏览器中打开 `index.html` 即可开始演示。

## 已包含页面

1. 启动页
2. 登录页
3. 注册/完善信息页
4. 首页
5. 穿搭详情页
6. 发布穿搭页
7. 标签选择页
8. 搜索页
9. 搜索结果页
10. 个人主页
11. 我的发布
12. 我的收藏
13. 关注列表/粉丝列表
14. 评论列表页
15. 点赞列表页
16. 商品跳转确认页
17. 设置页
18. 编辑资料页
19. 草稿箱页
20. 消息通知页

## 交互说明

- 首页卡片可进入穿搭详情
- 详情页可点赞、评论、收藏、分享，并跳转商品确认页
- 发布页可进入标签选择页并返回
- 个人主页可进入我的发布、我的收藏、编辑资料、消息通知
- 商品确认页可打开外部电商模拟窗口
- 全局提供返回、刷新、加载更多演示工具
