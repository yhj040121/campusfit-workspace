# 04 产品定位与价值主张

## 1. 产品定位

CampusFit 是一个面向大学生的校园穿搭内容与导购平台，以真实校园场景为内容组织方式，以社交互动为增长方式，以商品导购为转化方式。

一句话定位建议：

“帮助大学生在真实校园场景中，快速找到适合自己、适合预算、可直接购买的穿搭方案。”

## 2. 差异化定位

CampusFit 不等于：

- 传统商城
- 泛时尚内容社区
- 单纯的校园论坛

CampusFit 更像是：

- 场景化穿搭决策工具
- 内容驱动的导购入口
- 具备社区关系链的校园风格平台

## 3. 核心价值主张

### 3.1 对用户的价值

- 内容更真实：贴近校园，不是脱离日常的展示型内容。
- 选择更高效：通过场景、标签、预算快速定位内容。
- 决策更省心：从内容直接进入商品信息和购买入口。
- 社区更有归属感：可以关注同校或同风格创作者。

### 3.2 对创作者的价值

- 获得稳定曝光入口。
- 更容易沉淀粉丝和互动关系。
- 内容天然带有导购与合作延展空间。

### 3.3 对平台的价值

- 形成差异化内容壁垒。
- 建立“内容 + 社交 + 导购”的商业想象空间。
- 为后续品牌合作、专题投放和创作者运营提供基础。

## 4. 价值主张结构

| 维度 | 表达 |
| --- | --- |
| 目标人群 | 大学生、年轻校园用户、校园创作者 |
| 核心问题 | 缺少真实、可执行、低决策成本的校园穿搭参考 |
| 核心解决方案 | 场景化内容浏览、标签搜索、导购跳转、社交互动 |
| 差异化能力 | 校园场景标签、预算敏感内容、创作者导购结合 |
| 商业延展 | 商家合作、达人合作、专题活动、投放资源位 |

## 5. 品牌表达建议

- 关键词建议：真实、轻松、校园、可参考、可购买
- 内容基调建议：生活化、可信赖、非过度包装
- 视觉与交互方向建议：轻量、青春、明确标签导向

## 6. 不做什么

- 不做“所有人群通用”的泛时尚社区。
- 不做重交易闭环的综合电商。
- 不做复杂且门槛高的达人商业平台第一版。
