# 05 业务闭环与关键转化路径

## 1. 文档目标

明确 CampusFit 的业务如何跑通，以及产品中最重要的关键路径是什么。

## 2. 核心业务闭环

建议定义为：

`内容种草 -> 社交互动 -> 导购跳转 -> 数据回流 -> 内容优化 -> 用户复访`

各环节说明：

1. 内容种草：创作者或平台生产校园穿搭内容。
2. 社交互动：用户通过点赞、评论、收藏、关注表达兴趣。
3. 导购跳转：用户从内容详情进入商品信息或外部购买链接。
4. 数据回流：平台记录曝光、点击、互动、跳转等数据。
5. 内容优化：基于数据和标签体系调整内容分发与运营策略。
6. 用户复访：通过收藏、消息、关注内容更新形成回访。

## 3. 用户关键成功路径

### 3.1 普通用户路径

`进入首页 -> 浏览内容 -> 点击详情 -> 产生互动 -> 查看商品 -> 跳转购买 -> 收藏或关注 -> 后续复访`

### 3.2 创作者路径

`进入发布页 -> 上传内容 -> 打场景标签 -> 绑定商品 -> 提交审核 -> 内容发布 -> 获得互动 -> 查看数据反馈`

### 3.3 运营路径

`配置专题或推荐位 -> 审核内容 -> 监控数据 -> 调整运营策略 -> 触达用户或创作者`

## 4. 关键漏斗建议

### 4.1 用户消费漏斗

- 首页曝光
- 详情点击
- 互动发生
- 商品入口点击
- 商品跳转完成
- 回访行为发生

### 4.2 创作者供给漏斗

- 创作者注册
- 发起发布
- 完成内容编辑
- 通过审核
- 获得首批曝光
- 获得首批互动

## 5. 商业模式建议

当前阶段建议优先验证以下模式：

- 商品导购分佣
- 商家专题合作
- 校园创作者合作活动
- 运营资源位投放

## 6. 关键依赖

- 标签体系足够结构化
- 内容审核流程足够稳定
- 商品链接审核机制有效
- 数据埋点覆盖曝光、点击、互动和跳转

## 7. 风险点

- 内容供给不足会导致闭环中断。
- 商品链接质量不稳定会损害用户信任。
- 如果推荐和搜索弱，内容转化效率会显著下降。
- 缺少消息回流机制时，留存链条会变弱。

## 8. 阶段性判断标准

- 用户是否愿意持续浏览校园场景内容
- 创作者是否愿意稳定发布
- 内容是否能稳定触发互动和导购点击
- 平台是否能通过后台手段观察并优化上述链路
