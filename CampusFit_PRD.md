- 产品名称：CampusFit 校搭

## 1. 产品概述

### 1.1 项目背景

大学生群体具有较强的穿搭表达需求与社交分享需求，但在实际消费中普遍存在以下问题：

- 穿搭内容分散，缺少围绕校园真实场景的垂直内容平台
- 主流电商平台偏交易，缺少校园场景化搭配参考
- 主流内容平台偏种草，商品筛选、预算控制与导购链路不够聚焦
- 大学生预算敏感，更关注实穿性、复用率与理性消费

基于以上痛点，`CampusFit 校搭` 希望构建一个围绕大学生校园场景的内容种草、社交互动与导购转化平台，通过穿搭内容信息流、场景标签体系和商品导购能力，为大学生提供更贴近校园生活的穿搭解决方案。

### 1.2 产品定位

CampusFit 校搭不是传统自营商城，而是一个：

`以校园穿搭内容为核心、以社交互动为增长方式、以商品导购为商业转化手段的导购型电子商务平台。`

### 1.3 产品价值

对用户的价值：

- 提供校园真实场景下的穿搭灵感
- 帮助用户快速找到适合预算与风格的搭配
- 降低试错成本，提升购物决策效率
- 通过互动、关注、收藏建立社区归属感

对平台的价值：

- 通过校园垂直定位形成差异化内容壁垒
- 通过场景标签和预算标签提升导购效率
- 通过商家合作、导购分佣和达人合作实现盈利

## 2. 产品目标

### 2.1 用户目标

- 用户能快速浏览与自己校园生活高度相关的穿搭内容
- 用户能通过标签和搜索找到目标场景的穿搭方案
- 用户能从内容详情中直接获取商品购买入口
- 用户能完成发布、互动、收藏、关注等完整社交行为

### 2.2 业务目标

- 建立“内容种草 -> 社交互动 -> 导购跳转 -> 盈利转化”的业务闭环
- 提升内容消费深度与用户停留时长
- 提升商品导购点击率与转化率
- 搭建校园达人与品牌合作的商业化基础

### 2.3 毕设展示目标

- 页面完整
- 功能闭环清晰
- 商业模式可解释
- 用户场景明确
- 产品逻辑与电子商务属性成立

## 3. 目标用户与用户画像

### 3.1 核心用户

`18-24 岁在校大学生`

### 3.2 用户特征

- 注重穿搭表达与校园社交展示
- 对预算敏感，偏好高性价比商品
- 喜欢真实、可复用、可模仿的内容
- 愿意为“实用穿搭 + 省时决策”买单

### 3.3 用户画像

#### 用户画像 A：校园日常穿搭型

- 性别：女
- 年龄：20 岁
- 身份：大二学生
- 关注点：早八、图书馆、社团活动怎么穿
- 核心诉求：看真实校园穿搭案例，快速获取同款链接

#### 用户画像 B：预算敏感理性消费型

- 性别：不限
- 年龄：21 岁
- 身份：大三学生
- 关注点：预算 100-200 元内的高复用单品
- 核心诉求：减少冲动消费，优先考虑可复用、实穿的单品

#### 用户画像 C：校园内容创作者型

- 性别：不限
- 年龄：20-23 岁
- 身份：活跃穿搭分享者
- 关注点：内容曝光、粉丝互动、合作邀约
- 核心诉求：通过优质穿搭内容获得关注、合作机会和收益

## 4. 用户场景

### 4.1 典型使用场景

1. 用户早上出门前打开 APP，查看“早八课”穿搭推荐
2. 用户在图书馆或社团活动前搜索对应场景搭配
3. 用户看到喜欢的穿搭后，进入详情页查看商品信息并点击导购链接
4. 用户发布自己的穿搭内容，打上校园场景标签并挂载商品链接
5. 用户通过点赞、评论、收藏、关注参与社区互动
6. 创作者接收品牌合作消息，参与校园专题内容合作

### 4.2 用户旅程简述

`启动/登录 -> 首页浏览 -> 内容详情 -> 商品导购 -> 收藏/关注 -> 搜索/筛选 -> 发布内容 -> 获取互动/合作消息`

## 10. 盈利模式设计

本产品作为导购型电子商务平台，盈利模式不以自建支付交易为核心，而以内容导购和平台服务收益为核心。

### 10.1 商品导购佣金

模式说明：

- 用户浏览穿搭内容
- 点击商品链接跳转到外部电商平台
- 用户完成购买后，平台按成交金额获得一定比例导购佣金

适用价值：

- 与平台内容导购定位一致
- 电商属性明确
- 易于答辩说明商业闭环

### 10.2 商家推广位收费

模式说明：

- 商家可购买首页推荐位、场景专题位、标签专区位
- 平台向商家收取广告或内容推广服务费

适用价值：

- 适合平台早期变现
- 不完全依赖成交转化

### 10.3 校园达人合作服务费

模式说明：

- 平台连接品牌商家与校园穿搭达人
- 平台提供内容组织、专题投放、合作管理服务
- 平台收取服务费或抽取合作分成

适用价值：

- 强化社交属性与平台内容生态
- 有助于打造品牌专题、校园活动和商业合作场景

### 10.4 盈利公式

`平台收益 = 商品导购佣金 + 商家推广收入 + 达人合作服务费`

## 16. 风险与优化建议

### 16.1 当前潜在风险

- 若导购链接完全开放，可能出现低质量外链和广告刷屏
- 若平台只依赖导购佣金，早期流量不足时变现会较弱
- 若社交内容过度商业化，用户会降低社区信任

### 16.2 优化建议

- 引入合作商品标识和链接审核机制
- 强化“预算透明 + 理性消费”定位
- 用校园专题和品牌合作丰富商业化方式
- 建立内容质量评分与创作者等级体系

## 17. 结论

CampusFit 校搭是一款围绕大学生校园场景构建的穿搭社交导购型电子商务 APP。其产品核心不在于自建交易闭环，而在于通过真实场景内容、社区互动与商品导购能力，形成：

`内容分发 -> 社交互动 -> 导购跳转 -> 商业转化`

的完整电子商务逻辑。

从毕业设计角度看，该产品具备以下优势：

- 用户群体明确
- 使用场景真实
- 页面结构完整
- 业务逻辑闭环清晰
- 盈利模式可解释
- 适合高保真原型展示与产品答辩
